## Status Enums (exact string values)

**`content_ideas.status`:**
- `pending_validation` — just generated by AI, awaiting review
- `approved` — staff approved; DB trigger auto-creates a draft row in `content_queue_schedules`
- `rejected` — discarded, never enters calendar

**`content_queue_schedules.content_status`:**
- `draft` — auto-created by DB trigger on idea approval
- `scheduled` — staff set a publish time
- `uploaded` — media asset linked
- `published` — successfully posted to TikTok
- `failed` — publish attempt failed

**`content_assets.content_type`:**
- `poster_photo` — PNG/JPG; multiple allowed per schedule (carousel)
- `short_video` — MP4/MOV; max 50 MB

**`interaction_messages.channel_type`:**
- `comment` — TikTok comment
- `dm` — TikTok direct message

**`tiktok_accounts.connection_status`:**
- `connected`
- `disconnected`

**`classify_type_messages.priority_level`:**
- `high` | `medium` | `low`

**`classify_type_messages.sentiment_type_name`:**
- `purchase_intent` | `complaint` | `general_inquiry` | `compliment` | `spam`

## Time Vocabulary

| Term | Meaning | Where used |
|---|---|---|
| WIB | Waktu Indonesia Barat = GMT+7 | Display only |
| UTC | What we store | DB `TIMESTAMPTZ` columns |
| `jakartaTime.ts` | Server-side UTC↔WIB converter | Backend only |
| `formatDate.ts` | Client-side WIB display (dayjs) | Frontend only |
| `nowJakarta()` | Current time as WIB string | Backend |
| `jakartaToUTC()` | Convert user input WIB → UTC for storage | Backend |
| `formatWIB()` | Format UTC timestamp → WIB for display | Frontend |

Never say "Jakarta time" — say **WIB**. Never call `new Date()` without going through `jakartaTime.ts` on the backend.

## Backend Layer Vocabulary

| Layer | Folder | Responsibility |
|---|---|---|
| Routes | `src/routes/` | URL path + middleware chain |
| Middleware | `src/middleware/` | Auth, role gate, validation |
| Controller | `src/controllers/` | Parse req, call service, return response |
| Service | `src/services/` | Business logic, DB calls, external APIs |
| Model | `src/models/` | Supabase query wrappers |
| Job | `src/jobs/` | Cron tasks (auto-publish, fetch-interactions) |

## Frontend Layer Vocabulary

| Layer | Folder | Responsibility |
|---|---|---|
| Page | `src/pages/` | Route-level component, default export |
| Component | `src/components/` | Reusable UI, default export |
| Context | `src/context/` | Global state, **named** export |
| Service | `src/services/` | All API calls (axios), never in components |
| Hook | `src/hooks/` | Data-fetching + state logic for pages |
| Util | `src/utils/` | Pure helpers (formatDate, tokenHelper) |

## API Route Conventions

```
POST /api/auth/*          — authentication
GET|PUT /api/profile/*    — user profile
GET|POST|PUT|DELETE /api/calendar/*   — UC007
POST|GET|DELETE /api/media/*          — UC008
POST /api/prompt          — UC004
GET|POST /api/content/*   — UC005–UC006
GET /api/publish/*        — UC009
GET|POST /api/interaction/* — UC010–UC012
GET /api/dashboard/*      — UC013
GET|POST /api/tiktok/*    — OAuth + publish
POST /api/chatbot/message — AI chatbot
```

**Rule:** `VITE_API_BASE_URL` already ends in `/api`. Frontend service paths use `/prompt`, NOT `/api/prompt`.
